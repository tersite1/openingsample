# 오프닝 서비스 구현 플랜 (강남구 MVP)

## 1. 서비스 개요

**미션**: 점포 계약 이후 ~ 영업 시작까지의 오픈 준비 과정을 통합 지원

**타겟**: 강남구 5060 중장년층 생계형 창업자

**포지셔닝**: "대형 프랜차이즈처럼 편하게, 하지만 가격은 저렴하게"

---

## 2. 핵심 기능 구조

### Phase 1: MVP (현재 구현됨)
- [x] 회원가입/로그인 (이메일 인증)
- [x] 게스트 둘러보기
- [x] 오픈 패키지 매물 목록/상세
- [x] 3D 공간 배치 플래너
- [x] 견적 생성/저장
- [x] 상담 신청 (컨설팅 예약)
- [x] **가구 마켓** (중고 장비 거래)
- [x] 토스 결제 연동

### Phase 2: 핵심 기능 (구현 필요)
- [ ] **창업 체크리스트** (오픈까지 할일 관리)
- [ ] **업체 중개 시스템** (인테리어, 청소, 간판 등)
- [ ] **통합 견적 계산기** (초기비용 + 운영비)
- [ ] **강남구 상권 데이터** 연동
- [ ] **정부 지원사업 안내** (희망리턴패키지 등)

### Phase 3: 확장 기능
- [ ] 폐업자 → 창업자 매칭
- [ ] 실시간 진행상황 공유 (업체-고객)
- [ ] AI 기반 예상 매출 분석
- [ ] 타 자치구 확장

## 3. 데이터베이스 설계

### 현재 테이블
```
profiles          - 사용자 프로필
consultings       - 상담 내역
quotes            - 견적서
packages          - 오픈 패키지
furniture_listings - 가구 매물
payments          - 결제 내역
```

### 추가 필요 테이블
```sql
-- 1. 창업 체크리스트
startup_checklists (
  id, user_id, business_type,
  items JSONB,  -- [{task, category, due_date, completed, order}]
  created_at
)

-- 2. 업체 정보
vendors (
  id, name, category,  -- 인테리어, 청소, 간판, POS, 보험 등
  region,              -- 강남구
  contact, description,
  rating, review_count,
  price_range,
  portfolio_images[]
)

-- 3. 업체 견적 요청
vendor_quotes (
  id, user_id, vendor_id,
  request_details JSONB,
  quoted_price, status,
  created_at
)

-- 4. 정부 지원사업
government_programs (
  id, name, organization,
  target_audience,      -- 예비창업자, 폐업예정자 등
  support_amount,
  application_period,
  link, description
)

-- 5. 강남구 상권 데이터
gangnam_districts (
  id, dong_name,        -- 역삼동, 논현동 등
  avg_rent, avg_deposit,
  foot_traffic,
  popular_categories[],
  competition_level
)
```

## 4. 페이지 구조

1. **홈** - 추천 패키지, 최근 매물, 빠른 메뉴
2. **가구 마켓** - 중고 장비 거래
3. **패키지** - 오픈 패키지 목록
4. **내 상담** - 상담 내역
5. **더보기** - 설정, 지원사업 안내

### 추가 페이지 (Phase 2)
- **/checklist** - 창업 체크리스트
- **/vendors** - 업체 찾기 (카테고리별)
- **/calculator** - 창업비용 계산기
- **/support** - 정부 지원사업

## 5. 강남구 특화 데이터

### 동별 상권 특성
| 동 | 특성 | 추천 업종 |
|-----|------|----------|
| 역삼동 | 오피스 밀집 | 점심 한식, 카페, 편의점 |
| 신사동 | 가로수길 상권 | 카페, 브런치, 의류 |
| 논현동 | 유흥/먹자골목 | 주점, 고기집 |
| 청담동 | 고급 상권 | 파인다이닝, 럭셔리 |
| 삼성동 | 코엑스/오피스 | 프랜차이즈, 패스트푸드 |
| 대치동 | 학원가 | 분식, 편의점, 카페 |

### 평균 창업 비용 (강남구 기준)
- 보증금: 3,000~8,000만원
- 권리금: 5,000~2억원
- 인테리어: 평당 150~300만원
- 월 임대료: 200~500만원

---

## 6. 업체 네트워크 (확보 필요)

### 카테고리별 파트너
1. **철거/원상복구** - 2~3개 업체
2. **인테리어** - 업종별 3~5개 업체
3. **간판/사인물** - 2개 업체
4. **청소** - 1~2개 업체
5. **통신/POS** - KT, SKT 대리점 연계
6. **보험** - 화재/배상책임 보험사
7. **주류/음료 도매** - 지역 도매상

---

## 7. 수익 모델

1. **중개 수수료** - 업체 연결 시 수수료 (3~5%)
2. **가구 마켓 수수료** - 거래 성사 시 (5~10%)
3. **광고/노출** - 업체 프로필 상위 노출
4. **프리미엄 컨설팅** - 1:1 창업 코칭
